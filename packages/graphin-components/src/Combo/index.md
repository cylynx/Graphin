# Combo 分组节点

Combo 分组节点，是非常实用的一种图分析组件，可搭配其他组件，支持较复杂的功能，例如：带有节点分组的图布局，嵌套分组等。

## 功能特性

- 技术底座和 G6.Combo 保持一致
- 支持多个轮廓
- 支持用户自定义样式
