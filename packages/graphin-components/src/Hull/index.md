# Hull 轮廓包裹

Hull 轮廓包裹，是常见的一种图分析组件，可以将所属在一组的节点描边产生轮廓，从而在视觉上区分分组，轮廓组件不破坏原先的节点布局，是一种高效简单的分析组件

## 功能特性

- 技术底座和 G6.Hull 保持一致
- 支持多个轮廓
- 支持用户自定义样式
